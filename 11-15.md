# 11.按照给定数组，求最大容量
1. 设置左右双指针指向数组的两端
2. 计算此时的容量，如果大于最大值，更新最大值
3. 比较左右指针所指数字，向内移动较小值的指针，如果新值等于旧值，继续移动
```
  public int maxArea(int[] height) {
        int res = 0, left = 0, right = height.length - 1;
        while (left < right) {
            if (height[right] > height[left]) {
                res = Math.max(res, height[left] * (right - left));
                int cur = height[left];
                while (left < right && height[left] <= cur) left++;
            }
            else {
                res = Math.max(res, height[right] * (right - left));
                int cur = height[right];
                while (left < right && height[right] <= cur) right--;
            }
        }
        return res;
    }
```

# 12.int数字转换成罗马数字形式
1. 分别按照从大到小的顺序设置数字和罗马形式数组
2. 如果int数大于数字数组当前数字，int数减去当前数字，添加相应的罗马形式到结果字符串内
```
  public String intToRoman(int num) {
        String [] symbol = {"M", "CM", "D", "CD", "C", "XC", "L", "XL", "X", "IX", "V", "IV", "I"};
        int [] value = {1000, 900, 500, 400, 100, 90, 50, 40, 10, 9, 5, 4, 1};
        String str = "";
        for (int i = 0; num != 0; i++) {
            while (num >= value[i]) {
                num -= value[i];
                str += symbol[i];
            }
        }
        return str;
    }
```

# 13.罗马数字转换为int数字形式
1. 根据当前字符将相应的数字加入数组
2. 循环遍历数组，如果当前数字比下一个位置的数字小，就将结果减去当前数字，否则将当前数字加入结果
```
  public int romanToInt(String s) {
        int total = 0;
        int len = s.length();
        int[] store = new int[len];
        char[] arr = s.toCharArray();
        for (int i = 0; i < len; i++) {
            switch(arr[i]) {
                case 'M':
                store[i] = 1000;
                break;
            case 'D':
                store[i] = 500;
                break;
            case 'C':
                store[i] = 100;
                break;
            case 'L':
                store[i] = 50;
                break;
            case 'X' :
                store[i] = 10;
                break;
            case 'V':
                store[i] = 5;
                break;
            case 'I':
                store[i] = 1;
                break;
            }
        }
        for (int i = 0; i < len - 1; i++) {
            if (store[i] < store[i + 1]) {
                total -= store[i];
            }
            else { 
                total += store[i];    
            }
        }
        return total + store[len - 1];
    }
```

# 14.最长公共前缀
1. 取第一个字符串作为公共前缀
2. 遍历数组，判断公共前缀在当前字符串里的位置，如果不为0的话，就将公共前缀长度缩短一位，直到位置为0
```
  public String longestCommonPrefix(String[] strs) {
        if (strs.length == 0) return "";
        String prefix = strs[0];
        for (int i = 1; i < strs.length; i++) {
            while (strs[i].indexOf(prefix) != 0) {
                prefix = prefix.substring(0, prefix.length() - 1);
                if (prefix.isEmpty()) return prefix;
            }
        }
        return prefix;
    }
```

# 15.求数组里满足和为0的三个数字
1. 对数组进行排序
2. 如果当前字符和前一位字符相同，移动指针
3. 如果当前字符*3大于0，退出
4. 如果当前字符加上2倍的最后一个字符仍然小于0，移动指针
5. 新建两个指针指向当前字符下一个位置left和最后一个字符right，如果当前字符加上left和right指向数字的和小于0，移动left指针；如果和大于0，移动right指针；如果等于0，
```
  public List<List<Integer>> threeSum(int[] nums) {
        int len = nums.length;
        if (len < 3) return new ArrayList<List<Integer>>();
        Arrays.sort(nums);
        List<List<Integer>> ret = new ArrayList<>();
        for (int i = 0; i < len; i++) {
            while (i > 0 && i < len && nums[i] == nums[i - 1]) i++;
            if (i >= len - 2) break;
            if (nums[i] * 3 > 0) break;
            if (nums[i] + nums[len - 1] * 2 < 0) continue;
            int left = i + 1, right = len - 1;
            while (left < right) {
                int num = nums[i] + nums[left] + nums[right];
                if (num > 0) right--;
                else if (num < 0) left++;
                else {
                    List<Integer> list = new ArrayList<>();
                    list.add(nums[i]);
                    list.add(nums[left]);
                    list.add(nums[right]);
                    ret.add(list);
                    while (left < right && nums[left] == nums[left + 1]) left++;
                    while (left < right && nums[right] == nums[right - 1]) right--;
                    left++;
                    right--;
                }
            }
        }
        return ret;
    }
```
